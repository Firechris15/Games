# Tetris Game

A fully functional web-based Tetris game built with HTML, CSS, and JavaScript.

## Features

- 🎮 **Classic Tetris Gameplay** - All 7 Tetris pieces with authentic mechanics
- 🎯 **Score System** - Track your score, level, and cleared lines
- 👁️ **Next Piece Preview** - See what piece comes next
- ⏸️ **Pause/Resume** - Take a break anytime
- 📱 **Responsive Design** - Works on desktop and mobile
- 🎨 **Beautiful UI** - Modern gradient design with smooth animations
- ⚡ **Progressive Difficulty** - Game speeds up as you progress

## How to Play

1. Open `index.html` in your web browser
2. Use arrow keys to control the pieces:
   - **← →** - Move pieces left/right
   - **↑** - Rotate pieces
   - **↓** - Drop pieces faster
   - **Space** - Pause/Resume game
3. Complete horizontal lines to clear them and earn points
4. Game ends when pieces reach the top

## Game Rules

- Each cleared line adds to your score
- Multiple lines cleared at once give bonus points
- Difficulty increases every 10 lines cleared
- Game speeds up as you level up

## Controls

| Key | Action |
|-----|--------|
| ← → | Move left/right |
| ↑ | Rotate |
| ↓ | Drop faster |
| Space | Pause/Resume |
| Button | Pause Game |
| Button | Reset Game |

## Technical Details

- Built with vanilla JavaScript (no dependencies)
- Canvas API for rendering
- RequestAnimationFrame for smooth 60 FPS gameplay
- Responsive CSS Grid layout

## Scoring

- 1 line cleared: 100 points
- 2 lines cleared: 400 points
- 3 lines cleared: 900 points
- 4 lines cleared: 1600 points

Enjoy playing!
